# Pacman-Game
A)Features:

1.Pacman (C): Controlled with WASD keys

2.Ghosts (G): 4 AI-controlled ghosts that move randomly

3.Dots (.): Collect them for points (10 points each)

4.Walls (#): Obstacles that neither Pacman nor ghosts can pass through

5.Score tracking: Earn points by collecting dots

B)How to Play:

1.Use W/A/S/D keys to move Pacman up/left/down/right

2.Collect all the dots while avoiding the ghosts

3.Press X to quit the game

4.Game ends if a ghost catches you

C)To Compile and Run:

gcc pacman.c -o pacman
pacman.exe

Note:Only for Windows..
